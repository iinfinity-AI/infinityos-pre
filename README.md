# InfinityOS - Premium Pre-Launch Landing Page

## Tech Stack
- Nuxt 3 + Vue 3 + Tailwind CSS
- Fully responsive SaaS design
- Production-ready frontend only

## Quick Start
```bash
npm install
npm run dev
```

**Live:** http://localhost:3000

## Features
- Hero with dashboard mock
- Problem/Solution sections
- Modules showcase (20+ modules)
- Pricing tiers with expansion model
- Scaling timeline
- FAQ accordion
- All CTAs → Microsoft Forms waitlist

## Deployment
```bash
npm run generate
```

Static site ready in `.output/public`
